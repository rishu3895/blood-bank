Blood Bank Web Application 

Following functionalities are defined in the web application

1. There are two type of users (hospitals, receivers).
2. Different registration pages for different users.
3. A signle sign in page for users.
4. A hospital once logged in can edit the available blood quantity in their blood bank. 
    Also it can view all the different requests made by receivers for blood sample.
5. A receiver once logged in can request for any blood group that they require.
6. A single page to show all the blood groups available in all the blood banks. 

Technologies Used:
For the fronend I used HTML, CSS, Bootstrap, JavaScript, JQuery
For the server side I used Php
For the database I used MySQL
